```shell
tail -n 50 wx.log
```

示例：查看/var/log/boot.log，只显示最后一行。则执行

```shell
tail -n 1  /var/log/boot.log
```

```shell
tail -n 1000 #显示最后1000行
tail -n +1000 #从1000行开始显示，显示1000行以后的
head -n 1000 #显示前面1000行
```

实时监听文件末尾多行的变化，打印到控制台。ctrl+c 退出

```shell
tail -f file #默认显示10行
tail -500f file #调整为500行
```
